(async, static) filterProfesor(nombreopt, apellido1opt, sexoopt, departamentoopt) → {Promise.<Object>}

Filtra los profesores según los parámetros proporcionados (nombre, apellido1, sexo, departamento).
Parameters:
Name Type Attributes Description
nombre string <optional>
 El nombre del profesor a buscar. (opcional)
apellido1 string <optional>
 El primer apellido del profesor a buscar. (opcional)
sexo string <optional>
 El sexo del profesor a buscar. Debe ser "H" para hombre o "M" para mujer. (opcional)
departamento string <optional>
 El nombre del departamento al que pertenece el profesor. (opcional)
